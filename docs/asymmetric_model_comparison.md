# 非対称モデルとの比較

**目的**: 対称行列仮定 $A_{ij} = A_{ji}$ の妥当性を定量的に議論する。

---

## 1. 現状

- 変分原理（Lagrangian）から $A_{ij} = A_{ji}$ が帰結
- 生物学的には非対称もあり得る（例: A→B の促進と B→A の抑制が異なる）

---

## 2. 比較アプローチ

### 2.1 拡張モデル

$$A_{ij} \neq A_{ji} \quad \text{for some } (i,j)$$

- パラメータ数増加: 対称 15 vs 非対称 20 (上三角のみ) または 25 (全要素)
- 同定可能性がさらに悪化する可能性

### 2.2 検証方法

1. **BIC/AIC 比較**: 非対称モデルをフィットし、対称モデルと BIC で比較
2. **予測精度**: 両モデルで RMSE を比較
3. **事後幅**: 非対称成分 $A_{ij} - A_{ji}$ の事後が 0 を包含するか

### 2.3 実装

- `theta_to_matrices` を拡張: 上三角と下三角を別パラメータに
- 既存の locked indices を非対称版にマッピング
- TMCMC で推定し、$A_{ij} - A_{ji}$ の事後分布をプロット

---

## 2.4 対称性検証（実装済み）

現行モデルは $A_{ij} = A_{ji}$ を強制。検証スクリプト `tools/verify_symmetric_A.py` で全条件の $\|A - A^\top\|_F = 0$ を確認。

```bash
make verify-symmetric-A
```

---

## 3. 期待される結果

- 非対称成分が 0 に近い → 対称仮定で十分
- 有意な非対称性 → モデル拡張の正当化
- 同定不能 → 現行の対称モデルを維持
